Petunjuk Instalasi

  1. Tutup Cursor

    2. Linux

    1. Buka terminal
    2. Jalankan perintah berikut untuk mengunduh dan memasang fakem:
      curl -fsSL https://raw.githubusercontent.com/se7uh/cursor/refs/heads/main/new.sh -o ~/.local/bin/fakem && chmod +x ~/.local/bin/fakem
    3. Pastikan ~/.local/bin ada di PATH anda. Jika belum, tambahkan baris berikut ke ~/.bashrc atau ~/.zshrc:
      export PATH="$HOME/.local/bin:$PATH"
    4. Sekarang anda bisa menggunakan perintah fakem dari terminal manapun

Cara Penggunaan

fakem {on|off|new|gen|reset|run|update|version}
  • on Aktifkan fake machine ID
  • off Nonaktifkan fake machine ID
  • new Generate fake machine ID baru
  • gen Generate data telemetry baru dan update storage.json
  • reset Reset semua ID (machine ID dan telemetry)
  • run Jalankan aplikasi dengan fake machine ID
  • update Perbarui script ke versi terbaru
  • version Tampilkan informasi versi
